Accessibility

Accessibility information for MoofDash on iPhone.

Supported System Features

MoofDash is built with native iOS interface components where possible and is intended to work with standard iOS accessibility settings.

Recommended Settings

  • Use VoiceOver for spoken navigation where supported by iOS.
  • Use Display & Text Size settings to adjust contrast and legibility.
  • Use Reduce Motion if motion effects are uncomfortable.

Accessibility Feedback

If you find an accessibility issue in MoofDash, email Perez.Cristian+ace@ace.aaa.com with the device model, iOS version, and the screen or feature where the issue occurs.
